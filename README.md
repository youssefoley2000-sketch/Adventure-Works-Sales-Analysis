# Adventure Works Sales Analysis Dashboard

## Overview
This repository contains an interactive Power BI dashboard built using the Adventure Works dataset.
The dashboard provides insights into sales performance, profitability, and trends across time,
weekdays, and countries to support data-driven decision-making.

## Key Metrics
- Total Quantity Sold: 139.40K
- Total Revenue: $95.22M
- Total COGS: $56.52M
- Total Profit: $38.70M
- Profit Margin: 40.6%
- Total Transactions: 13.35K

## Analysis Highlights
- 2007 is the highest-profit year.
- Q2 is the strongest quarter, contributing 29% of total profit.
- Weekdays generate 72.2% of total profit.
- Wednesday to Friday account for 45.7% of total profit.
- Thursday is the top-performing day.

## Dashboard Features
- KPI cards for high-level performance overview
- Interactive slicers (Year, Month, Country)
- Revenue / Profit / Transaction selector
- Monthly, quarterly, and weekday trend analysis
- Country-level performance comparison
